Block-Level Elements List

  1. h1 to h6 (Headings 1 to 6)
  2. p (Paragraph)
  3. hr (Horizontal Line Breaks)
  4. blockquote (A section that is quoted from another source)
  5. ol (Odered Lists)
  6. ul (Unordered Lists)
  7. li (Lists)
  8. dd (Describe a term/name in a description list)
  9. dt (Defines terms/names)
  10. dl (Description list)
  11. table (Table define as rows and columns)
  12. tr (Table row)
  13. th (Table header)
  14. td (Element defines a table cell)
  15. form (To create an HTML form for user input)
  16. figure (Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.)
  17. figcaption (Element to define a caption for the photo)
  18. div (Block-Level Container)
  19. footer (A footer for a document or section of web page)
  20. header (Element represents a container for introductory content or a set of navigational links)
  21. artrical (Independent, self-contained content)
  22. address (The contact information for the author/owner of a document or an article in italic form)
  23. aside (Content is often placed as a sidebar in a document)
  24. canvas (It is used to draw graphics, on the fly, via scripting "usually JavaScript")
  25. fieldset (It is used to group related elements in a form)
  26. main (element should be unique to the document. There must not be more than onemain element in a document)
  27. nav (A set of navigation links)
  28. noscript (An alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script. It can be used in both head and body)
  29. pre (Preformatted text)
  30. section (Defines a section in a document)
  31. tfoot (It is used to group footer content in an HTML table)
  32. video (It is used to embed video content in a document, such as a movie clip or other video streams)


Inline Elements List

  1. a (Anchor)
  2. b (Bold Text)
  3. br (Single Line Breaks in a text)
  4. abbr (Abbreviation)
  5. acronym (Tag was used in HTML 4 to define an acronym.Not Supported in HTML5)
  6. bdo (Stands for Bi-Directional Override. Tag is used to override the current text direction)
  7. cite (The Title of a work)
  8. code (Tag is used to define a piece of computer code in monospace font)
  9. dfn (Tag stands for the "definition element")
  10. strong (Important Text)
  11. em (Emphasized Text)
  12. i (Italic)
  13. q (Quotation Mark)
  14. sub (Subscript Text)
  15. sup (Superscript Text)
  16. button (Clickable Button)
  17. img (Image)
  18. label (A Label for several elements)
  19. input (An Input field where the user can enter data)
  20. textarea (To collect user inputs like comments or reviews)
  21. span (Inline Container)
  22. kbd (Tag is used to define keyboard input in monospace font)
  23. map (An image map, with clickable areas)
  24. object (Tag defines a container for an external resource)
  25. output (Tag is used to represent the result of a calculation) (like one performed by a script)
  26. script (Tag is used to embed a client-side script) (JavaScript)
  27. select (Element is used to create a drop-down list)
  28. small (Define a smaller text)
  29. time (Tag defines a specific time or date)
  30. tt (Tag was used in HTML 4 to define teletype text. Not Supported in HTML5)
  31. var (Tag is used to defines a variable in programming or in a mathematical expression)
